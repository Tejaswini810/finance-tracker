# 💰 Premium Finance Tracker

A modern personal finance tracking web application built with React.js that allows users to manage income and expenses efficiently.

## 🚀 Features

* Add Income and Expense Transactions
* Delete Transactions
* Filter Transactions by Type, Category, Date Range
* Dashboard Summary
* Pie Chart and Bar Chart
* Dark Mode Toggle
* Local Storage Persistence

## 🛠️ Tech Stack

* React.js
* JavaScript
* CSS3
* Recharts
* Local Storage API

## 📦 Installation / Setup

1. Clone repository
2. Run npm install
3. Run npm install recharts
4. Run npm start

## 📋 Assumptions / Trade-offs

* Data stored in local storage
* No backend database used
* Single-user application
